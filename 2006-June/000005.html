A new RACK (level 2) automake template has been added to the svn repository.
The projects created with this template depends on RACK level 1 projects
(we call it 'level 2' projects).
You can get it with:

svn co svn://svn.berlios.de/rack/automake/level2

Please have a look at the README files in the level2 root directory.
